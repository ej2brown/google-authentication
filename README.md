## Setup
**⚠️ This will not work on non-whitelisted URL's! ⚠️**

1. [Obtain an client id](https://developers.google.com/) 
2. Put it in the .env file
3. Install dependencies with `npm install`.

## Running Webpack Development Server

```sh
npm start
```
